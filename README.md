# Bersih Itu Patriotik — Vanilla Website (HTML/CSS/JS)

## Cara pakai
1. Simpan file-file:
   - index.html
   - styles.css
   - scripts.js
   - poster-a4.html
   - print.css
   - README.md

2. Buka `index.html` langsung di browser (double-click), atau di VS Code gunakan Live Server.

3. Statistik (Chart.js) sudah terintegrasi via CDN. Form laporan disimpan di `localStorage` (demo).

## Kustomisasi cepat
- Ganti hero image di index.html (tag <img>) jika mau pakai gambar lokal.
- Untuk menyimpan laporan ke server, ganti fungsi `saveReport` di scripts.js agar memanggil API.
- Untuk cetak poster, buka `poster-a4.html` → Print → pilih A4.
